# Limitations

- It uses fake sample data.
- It uses deterministic heuristics, not a paid LLM.
- A human planner should review the final schedule.
- It is a prototype, not a production WFM optimizer.
- Real deployment would need integrations with HR systems, forecasting tools, and compliance rules.
- It does not model all local labor law, union, or contract constraints.
- It does not optimize for cost, shrinkage, adherence, skill-level routing, or long-term employee preferences.
